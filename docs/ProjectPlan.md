# Job Tracker

## Overview

Job Tracker is a full-stack web application designed to help users organize and manage their job search.

Instead of keeping track of applications in spreadsheets or notes, users can manage companies, resume versions, job applications, interviews, and view useful statistics through an interactive dashboard.

The project is being built to strengthen my understanding of PostgreSQL, SQL, FastAPI, REST APIs, React, and full-stack software development while solving a real-world problem.
