# User Stories

## User Profile

### Story 1

As a user, I want to edit my personal information so that my profile stays up to date.

---

## Resume Management

### Story 2

As a user, I want to upload a new resume so that I can select the correct resume version when applying.

### Story 3

As a user, I want to edit an existing resume so that I can keep my resume information current.

### Story 4

As a user, I want to delete a resume that I no longer use.

---

## Company Management

### Story 5

As a user, I want to add a company that I'm interested in so that I don't have to manually type the company every time I apply.

### Story 6

As a user, I want to edit company information if it changes.

### Story 7

As a user, I want to delete companies that I no longer need.

---

## Job Applications

### Story 8

As a user, I want to create a job application so I can track my job search.

### Story 9

As a user, I want to edit an existing application if information changes.

### Story 10

As a user, I want to delete an application.

### Story 11

As a user, I want to favorite important applications so I can easily find them later.

### Story 12

As a user, I want to update the application's status so I know where I am in the hiring process.

### Story 13

As a user, I want to search my applications so I can quickly find one.

---

## Interviews

### Story 14

As a user, I want to schedule an interview for an application.

### Story 15

As a user, I want to edit interview information.

### Story 16

As a user, I want to delete an interview.

---

## Dashboard

### Story 17

As a user, I want to see how many jobs I've applied to.

### Story 18

As a user, I want to see how many applications are favorited.

### Story 19

As a user, I want to see upcoming interviews.

### Story 20

As a user, I want to quickly view my most recent applications.
